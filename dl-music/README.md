# DL-MUSIC 

Downloads music to your Music directory. 

Usage: 
	-h : display this
	-u : url                required 
	-a : author / band      recommended
	-t : song title         recommended
	-l : album name         optional 
	-n : tracknumber        optional


Dependencies
- yt-dlp (downloading music)
- kid3-cli (open source audio file tag editor)
