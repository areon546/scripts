# makefile 

Basic command that adds a help template based on an
[auto documented makefile](https://marmelab.com/blog/2016/02/29/auto-documented-makefile.html)
article.


Tips and tricks:
\@ to make commands not be printed 
\# to comment commands 
\- [apparently](https://stackoverflow.com/questions/2670130/make-how-to-continue-after-a-command-fails) makes the script continue even through a fail, although unknown 
\&\& chains commands, eg cd dir1 && ./script.sh
